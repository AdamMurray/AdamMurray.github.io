NASA APOD

I started working on this website because I am a big fan of astronomical photography, and NASA's Astronomy Picture of the Day (APOD) website in particular; however, I find the current site to be very much lacking in appearance and usability. Therefore, after discovering about the recent open APIs which NASA have been working on, including the APOD API, I decided to create a site of my own which would be attractive, responsive, performant, and just nice to use. What I have created so far is not a finished product, but it does demonstrate what I am trying to achieve.

Mobile Menus

The following two pens are experiments in creating mobile menus which are comfortable to use, from a user experience standpoint.

Many current web and native mobile applications use the now ubiquitous "burger menu" as the primary means of site navigation. Frequently, this menu is placed either at the top-left (more common) or top-right of a page. This positioning creates an uncomfortable experience for the one-handed mobile user, due to the stretch required to reach the top parts of screen. Therefore, I decided to try to improve this situation with the menus below.

See the Pen UX Menu Idea #2 by Adam Murray (@AdamMurray) on CodePen.

CSS Raygun

This raygun was created using pure CSS. Each component that the gun comprises is a styled div element. The only JavaScript used is the trigger click handler which causes the shooting animation, and the beam colour toggle which is invoked by clicking on the light at the rear of the gun.

See the Pen Raygun by Adam Murray (@AdamMurray) on CodePen.

CSS Chemical cabinet

This pen is an experiment in creating a pure CSS cabinet, including items to appear on the shelves of the cabinet. The unit, shelves, shading, doors, etc, are all styled div elements, and the only JavaScript is a click handler attached to the doors which adds an active class an causes the doors to open.

See the Pen Chemical Cabinet by Adam Murray (@AdamMurray) on CodePen.
